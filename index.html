snow-fall by wgpsutherland

Setup:

All you need to do is include snowFall.css, snowFall.js and triangle.PNG in your project, and create a div with class="snow-field" on your webpage.

Change height and colour of div:

You may want to dive into snowFall.css and change the height and colour of the div. The following highlighted values are the ones you will need to change: 

.snow-field {
    background-color: rgb(40,40,40);
    height: 500px;
    position: relative;
    overflow: hidden; 
}
...
@-webkit-keyframes falling {
    ...
    100% {
        -webkit-transform: translate(0,600px) rotateY(720deg) rotateZ(480deg);
    }
}
@keyframes falling {
    ...
    100% {
        transform: translate(0,600px) rotateY(720deg) rotateZ(480deg);
    }
}
As you can see, the pixel values in the keyframes are 100 more than the height of the div. This is to accomodate for the height of the triangle.
